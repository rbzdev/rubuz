# Rubuzfolio - Portfolio Personnel

Bienvenue dans mon portfolio personnel ! Je suis un développeur Fullstack Web et Mobile, passionné par la création d'expériences utilisateur performantes et élégantes.

## 🚀 Stack Technique

- **Framework**: [Astro](https://astro.build/) (Web)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/) (React components)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Iconify](https://iconify.design/)
-  **Theming**: Dark mode support
- **I18n**: Multilanguage support (FR/EN)

## 🛠️ Structure du Projet

```text
/
├── src/
│   ├── components/  # Composants réutilisables
│   ├── layouts/     # Mise en page (Layouts)
│   ├── pages/       # Routes principales
│   ├── styles/      # Fichiers CSS et thèmes
│   └── lib/         # Utilitaires et hooks
└── public/          # Assets statiques
```

## 🧞 Commandes

| Commande           | Action                                         |
| :----------------- | :--------------------------------------------- |
| `pnpm install`     | Installe les dépendances                       |
| `pnpm dev`         | Démarre le serveur de développement            |
| `pnpm build`       | Build pour la production                       |
| `pnpm preview`     | Preview de la build locale                     |
